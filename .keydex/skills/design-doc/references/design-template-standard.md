# DES 文档模板（增强版）

本文档为 `/design` 命令的增强版模板。

目标不是只补齐章节，而是生成一份：
- 先看得懂整体架构与关键链路
- 再看得懂每个核心功能点如何落地
- 最终可直接进入开发、拆解与评审的 DES
